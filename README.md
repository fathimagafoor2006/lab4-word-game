**Lab4 Word Game**

- **Description**: A small lab repository. Includes a simple recursive `fibonacci(n)` example in `main.py` and supporting project files.

**Getting Started**
- **Requirements**: Python 3.8 or newer.
- **Run**: Execute the demo CLI which prints Fibonacci numbers.

```bash
python main.py 10
```

**Files**
- **`main.py`**: Recursive `fibonacci(n)` implementation and CLI runner.
- **`.gitignore`**: Common Python and editor ignores.
- **`JOURNAL.md`**: Agent activity log and change history.
